**1. Acesse:** https://registro.br

### Passo 2: Configurar DNS (Após Comprar)

Depois de registrar o domínio, configure para apontar para seu site:

#### No Painel do Registro.br ou seu provedor:

**1. Acesse o painel de DNS**

**2. Adicione os registros:**

**OPÇÃO A - CNAME (Mais Simples):**
```
Tipo: CNAME
Nome: @
Valor: lead-capture-56.stage-preview.emergentagent.com
TTL: 3600
```

```
Tipo: CNAME
Nome: www
Valor: lead-capture-56.stage-preview.emergentagent.com
TTL: 3600
```

**OPÇÃO B - A Record (Se souber o IP):**
```
Tipo: A
Nome: @
Valor: [IP do servidor Emergent]
TTL: 3600
```

**3. Salve as configurações**

**4. Aguarde propagação:** 4-48 horas (geralmente 4-6 horas)

### Passo 4: Certificado SSL (HTTPS)

O Emergent geralmente configura SSL automaticamente.

Se não, use **Cloudflare** (gratuito):

**1. Criar conta:** https://www.cloudflare.com

**2. Adicionar site:**
- Digite: quadra500.com.br
- Clique em "Add site"

**3. Cloudflare fornecerá nameservers:**
```
Exemplo:
ns1.cloudflare.com
ns2.cloudflare.com
```

**4. Atualize nameservers no Registro.br:**
- Vá no painel do Registro.br
- Mude os nameservers para os da Cloudflare

**5. Configure SSL:**
- No painel Cloudflare
- SSL/TLS → Full
- Always Use HTTPS: ON
